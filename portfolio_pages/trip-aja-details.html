Story Behind Development

To be honest, this application have been builded when i still in Bangkit Program, but that was still in Kotlin version with the application name. is "Heal & Go". So, when i come back to my study in university, i want to builded that in Flutter version to complete my subjects about Mobile Programming in my study with my beloved friends in university. To convert this program into Flutter, effectively takes 2-3 weeks to completed this. But, because i with my friends didn't take only Mobile Programming subjects in this semester, the development for this program take until 2 months including backend implementation to the application. When we developed this program to Flutter, we found some problem like overflow page in the application UI, misscolored the background, problem to implemented MVVM into this application, and problem about where the Machine Learning model can be stored to called by the application because this application needed Machine Learning to run the recommendation.

After exhausting 2 months, we can completed this application in Flutter version and test that to public, and with Usability Testing, our application can reach 89%, which means our application is so good to used by public. Even so, more people give the suggestion about to include localization or Indonesia language to this program because this program only have English language, and then give more content about the destination detail to the application. You can try this application in this page.

In this application, that have some pages and features, such as:

  • Splashscreen, you can find this when you start the application.
  • Onboarding, you can find this when you use the application for the first time.
  • Authentication, the application provides authentication to use main features in this program, such as Register and Login.
  • Homepage, you will find your top recommendation list in this page based on your preference and app's recommendation.
  • Discover, you will find all of destination list in this page, and you can find the destination based on search by keyword or category.
  • Profile, you can see your detail information in this page, and you can go out from main application to Login page using Logout button in this page.
  • Questionnaire, If you want to get your recommendation, you must answer all of the questions in this features.
  • Card Recommendations, after you answer all of the questions, the application will give you some recommendations about destination in the Card Stack form. You can like or dislike all of the recommendations that given by the app using the button or swipe the card.
